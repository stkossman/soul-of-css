This framework would not be possible without a NES.css since it inspired me to create similar framework but in another style.

For now, I did not manage to overcome and do better than NES.css, (even this page is not that good compared to my other works) but I'll try my best and will be updating this framework until it satisfies me. Maybe it is not perfect nor good because of lack of my knowledge in SCSS, creating frameworks etc., but anyway it worthed a try. Also, I know that this framework is not really useful, but I just wanted to create something that would help me in first place deal with my depression, and learn something new about CSS and SCSS.
